# ReadmitRadar
AI-powered hospital readmission risk scorer for discharge planning.

## Live App
[Click here to open]([your-streamlit-url](https://readmit-radar.streamlit.app/))

## What It Does
- Scores 30-day readmission risk from patient profile
- Generates AI clinical assessment using Claude
- Compares patient risk to national CMS benchmarks
- Provides specific discharge recommendations

## Tools Used
Python, Streamlit, Anthropic Claude API, Plotly

## Disclaimer
Educational portfolio project only. Not for clinical use.
